programming

I am a self-taught programmer. I've used a variety of resources to learn with, but freecodecamp.com has been the most useful. I earned my front end certificate from freeCodeCamp in early September 2017.

Through freeCodeCamp, I studied HTML5, CSS3, JavaScript, jQuery, and Bootstrap. I also learned a bit about algorithms.

I got started with code in late 2014 through Invent with Python, and from there I experimented with the Unity game engine, as well as Lua and love2d. I also started using git during this time.

These days, I'm learning React.js and Ruby on Rails.

My preference is for GNU/Linux operating systems (Arch, Linux Mint), but I'm flexible.
